## Overview

A simple, elegant digital tasbih counter for counting dhikr and prayers, built with Kotlin and Jetpack Compose. Completely offline with no data collection, keeping your spiritual practice private.

## Features

- **One-Tap Counting:** Simple, clean tap-to-count interface.
- **Reset Functionality:** Reset the counter instantly.
- **Completely Offline:** No internet connection required.
- **No Permissions Required:** No data collection, fully private.
- **Islamic-Inspired Design:** Clean UI tailored for daily dhikr.

## Tech Stack

- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Min SDK:** 29 · **Target SDK:** 36
- **Build Tool:** Gradle (Kotlin DSL)
